Fusion ICA Toolbox (FIT)

 

Introduction to FIT

Welcome to the Fusion ICA Toolbox (FIT). HTML help is provided to run the analysis quickly. For a detailed explanation of the toolbox you should refer to the manual. FIT is an application which runs on MATLAB 6.5 and higher. FIT implements the joint ICA ([3]) and parallel ICA ([1]) methods to examine the shared information across different modalities.
