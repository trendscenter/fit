Optimal Features

Before running this utility, set variable OPTIMIZE_FEATURES in 'ica_fuse_defaults.m' to 'Yes' and run joint ICA analysis. This will run ICA on different combinations of features in order to identify which combination of features most differentiates the groups. We use fMRI-fMRI example data-set to demonstrate the optimal features. After you click "Optimal Features" in "Utilities" drop down box, figure 1 will open. The parameters in the figure 1 are as follows:
  • "Select sorting criteria" - "Best" component is determined based on the sorting criteria selected. For spatial divergence sorting criteria information from "Select type of histogram" is used.
  • "Select type of histogram" - There are two options like "Feature" and "Component".
    • Feature - "Best" component voxels are sorted in descending order and Z-threshold is applied. This set of voxels is used as a mask to the original data. Cross-task histogram is constructed for each selected group.
    • Component - "Best" component voxels are sorted in descending order and cross-task histogram is constructed for each selected group.
  • "Select Groups" - Select at most two groups.
  • "Select Z threshold" - Z-threshold applied on "Best" component when "Feature" histogram is used.
When you click Calculate button spatial divergence between the selected groups is calculated using the cross-task histograms. The default divergence criteria is "renyi" with alpha = 2. Figure 2 shows the features ranked in descending order of their divergence.
