Setup Analysis For Parallel ICA

We use fMRI-gene data set to explain parallel ICA. When you click Setup Analysis button, a figure window will open to select the output directory for parallel ICA fusion. Figure 1 shows the initial parameters select window. The parameters in the figure 1 are as follows:
"Enter Name(Prefix) Of Output Files"
All the output files will be saved with this prefix.
"Have You Selected The Data Files?"
When you click Select button, data can be selected for parallel ICA fusion analysis. After the data is selected, a drop down box will be plotted with options as "Yes" and "No". If you select "No", data can be selected again.
"What Mask Do You Want To Use?"
There are two options like 'Default mask' and 'Select Mask'. Each option is explained below:
'Default Mask'
For fMRI and sMRI, the default mask includes only the voxels that are non-zero and not Nan for all the subjects. For gene data all indices are included.
'Select Mask'
You can select a mask file for fMRI and sMRI data. For SNP data, you need to enter the indices that will be included for the analysis.
We used "Select Mask" option and selected file 'myMask_t3.img' as mask for fMRI data and all SNP indices for gene data.
"Number Of PC for modality fMRI"
Enter number of principal components to be extracted from the data for feature "fMRI".
"Number Of PC for modality Gene"
Enter number of principal components to be extracted from the data for feature "Gene".
"Select Type Of Parallel ICA"
There are two options like "AA" and "AS". The "AA" option uses correlation measure between mixing coefficient of modality 1 with mixing coefficient of modality 2 whereas "AS" option uses correlation measure between mixing coefficient of modality 1 with source of modality 2.
"Select type of PCA"
There are two options like "Reference" and "Standard". "Reference" option uses information from groups to project eigen vectors to that dimension. Please see [1] for more information. Figure 3 will open when you click on "Reference" option. We entered a value of -1 for SZ group and a value of 1 for healthy group.
"Number of times ICA will run"
Number of times you want ICA to be run on the data.
Figure 4 shows the completed parameters for the analysis. When you click Done button, parallel ICA options will be shown in an input dialog box (Figure 4). After the parallel ICA options are selected, a parameter file with suffix "para_ica_fusion.mat" will be saved in the analysis output directory.
