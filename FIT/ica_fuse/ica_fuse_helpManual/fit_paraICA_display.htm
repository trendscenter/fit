Display Parallel ICA Components

Figure 1 will open after you click Display button and had selected the parameter file. The parameters in the display GUI are as follows:
  • "Comp No:" - Component numbers to display. This is dependent on the feature you have selected in the feature listbox. This option will be disabled when you sort components.
  • "Feature:" - You can select at most one feature to display. This option will be disabled when you sort components.
  • "Do You Want To Sort Components?" - You have the option to sort components based on correlation. When you select "Yes" for sorting components and click Display (Figure 1) button, components will be sorted based on the correlation. Figure 2 shows the results of sorting. When you click on any axes, enlarged view of that plot is shown in a different figure (Figure 3).
  • "Display Defaults" - Spatial map defaults will be shown in a figure window when you click on "Display Defaults" menu.
  • "Options" - There is an option to select the SNP's locus file. We have included locus file "LocusNames.txt" for the fMRI-gene example data set. Dominant SNP's will be printed to a text file having suffix "dominant_snps.txt" when components are sorted based on correlation. Dominant SNP's are determined based on Z-threshold for SNP's. You can change Z-threshold by changing variable SNP_Z_THRESHOLD in "ica_fuse_defaults.m" file.
