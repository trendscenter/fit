Joint ICA Toolbox

When you click Joint ICA button, joint ICA toolbox (Figure 1) will open. Joint ICA toolbox can also be opened by typing fusion('jointICA') at the MATLAB command prompt.

Figure 1: Joint ICA Toolbox

Joint ICA toolbox contains user interface controls like Setup Analysis, Run Analysis, Fusion Info, Display and "Utilities". These can be classified as analysis and display steps. We next explain Setup Analysis.
