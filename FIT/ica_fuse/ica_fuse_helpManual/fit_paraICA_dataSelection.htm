Data Selection For Parallel ICA

When you click push button Select in Setup Analysis, figure 1 will open to enter the number of groups for the analysis. After entering the information in figure 1, figure 2 will open to enter the names of groups and features. For the example data-set, the files filter pattern is the same for both groups. Enter only the file pattern (Figure 3) for features of first group. Since the directories where the groups are stored are separate for fMRI-gene data-set, select "No" for question "Is the data organized in one group folder?" This will let you select directory (Figure 4) for each group. After you complete the data selection, files selected for analysis are printed to a text file with suffix "para_ica_sel_data.txt".
