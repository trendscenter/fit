Display Joint ICA Components

After the analysis is complete, Display GUI (Figure 1) will open automatically unless you set OPEN_DISPLAY_WINDOW variable in 'ica_fuse_defaults.m' to 0. The parameters in the figure 1 are as follows:

  • "Comp No" - Select components of interest. By default, all components are selected.
  • "Feature" - Select features of interest. By default, all features are selected.
  • "Do you Want To Sort Components?" You have the option to classify components based on two sample t-test between loading coefficients of groups or spatial divergence between groups. More Information is given under "Sorting Components" section.
  • "Display Defaults" - Display defaults for spatial maps will open when you click "Display Defaults" menu.
  • "Utilities" - "Utilities" menu can be used to create ERP-fMRI movie when EEG-fMRI fusion analysis is run.
